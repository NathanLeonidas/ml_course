# Hands-on Machine Learning en Python

Ce dépôt contient un ensemble de travaux pratiques sur le **Machine Learning en Python**, utilisant des librairies comme **PyTorch** et **scikit-learn**.  
Il sert de complément au cours de Machine Learning à mon école et couvre des notions d'apprentissage supervisé, non supervisé, vision par ordinateur, et plus encore.
